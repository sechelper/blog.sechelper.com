加入社区

核心社区从最开始的几人，到现在的近百人，越来越多志同道合小伙伴加入了社区，感谢大家对助安社区的一切努力。

在这个信息嘈杂流量为王的时代，以免费资料分享加好友推课屡见不鲜，你们可能接触不到真正的自学平台，全都是一些xxx培训，致力于成为xxx，这和学员有半毛钱关系，来你这学习为了实用工作，不是看炫技的。买课或者线下培训的同学们注意选择，不要被一些名头忽悠了，学完发现什么干货都没有，还在工作边缘苦苦挣扎。别盲目买课，被所谓的优惠蒙蔽双眼，买之前多找些同行业人交流。

在这里忍不住吐槽一下，从始至终社区内都没有私信推过课，建立社区初期也给大家很多规划指导，小伙伴们问了一天的问题，君师也没有推过课或星球。免费的才是最贵的，没人指导你走一次弯路的时间需要多少钱？自己想吧。
期望大家也能够在社区里多交流分享，这里的分享指的不是分享多么高深的技术给大家，哪怕群里稍微指点一句话、交流个观点或者分享个资讯也行。社区成长起来大家才能获取到更多优质学习资源。社区内部干货的分享以后会趋于向有贡献的人分享。

我们欢迎更多热衷分享、热爱学习的小伙伴们加入。希望助安社区能成为自学安全的灯塔，让大家找到自己的前进方向。还是那句话我来掌舵，目标星辰大海。

¶监督学习

为了让社区内小伙伴学习真是操碎了心，大锅炖的方式大家都不学习，只能定向监督拉，大家就像小毛驴不打不走那种。

首先社区内分为几个小分队，还有其它俩神秘分队，进入社区就知道拉。

¶内部分享

社区内鼓励大家积极分享学习成果，不要觉得吃亏的是你，分享出来师傅们才能给你指正，也能促进大家交流，当然最最最重要的是有奖励。

别的社区是不是都是一群大佬在分享各种骚操作，但是想过没有你本身具备这个能力，只是没有被激发出来，授人以鱼不如授人以渔，看别人的文章学习的东西还能有自己写一遍多？。

¶B站公开课完全版

公开课已经出了几个系列课程，全免费，内部分享完全版

¶学习平台

平台也在建设中，安全学习孤立化严重，社区立志做一个安全学习百科wiki，就靠大家啦，一个人不是社区，一群人才是。

自学的微信公众号也在建设中，以后只要订阅就能写作业，看课程大纲，参与内部活动

大家肯定记不住各种安全学习域名，不要紧收藏咱们社区的导航页就够了

¶漏洞靶场

线上和线下靶场正在建设中，随着积累越来越多学习越方便。不限于复现漏洞，会增加很多场景渗透实战。

¶其它资源

君师会收集各种适合大家学习的资源，并且给大家一些建议。跟着学多交流，找不到工作才怪，多动脑勤动手，加入社区不可能找不到工作。

¶奖励机制

再看下奖励机制，只要你能坚持学习，星球的钱原路返回，来挑战？（得意的笑），欢迎咨询。

¶加入我们

纠结是否要购买的朋友，可以加【君师坤哥】微信 ，备注 “了解星球” ，给你最中肯的意见。

❤️考虑到学生党生活比较拮据，每位粉丝加入社区有一次 制定目标的机会 ，完成后享受半价到全价的退款（需扣除手续费，最终解释权由助安社区所有， 防止恶意薅羊毛）。目标可以是考研、进大厂、进某某公司、拿到满意薪资的 offer、考有些难度的证书（CISP-PTE、CISSP、OSCP…）等。

¶我适合加入么？

🌟拒绝躺平，人人有责，想要躺平进入星球也学不到任何知识，哪里都只能让你花冤枉钱。

以下几种情况的粉丝，非常欢迎加入：

  1. 大学生（大一或即将毕业）/社招找工作/转行等，感觉在网络安全行业迷茫没有前进方向
  2. 在职没机会实战，想提高实战攻防能力
  3. 觉得自己学习难，需要一个稳定圈子和小伙伴
  4. 想与圈内大师傅交流分享最新攻防经验
  5. 喜欢创作记录个人成长经历
  6. 想要在公司更近一步，社交账号获得更多小迷弟、小迷妹
  7. 想 “偷懒” 省事，领取一些成长资料、内部原创文章

以下情况，不适合加入：

  1. 不喜欢圈子交流学习
  2. 宣传自己的圈子，想拉人出逃
  3. 只希望在星球中获得听着高深莫测，实际没软用的技术知识
  4. 不能持续学习，只是想看热闹

社区发展方向以渗透为主，但社区不限于渗透，会涉猎代码审计、安全运维和安全开发等。希望能建立一个多元化交流平台，粉丝们有需求【狗头君师】会邀请好友讲解其它方向（二进制、CTF、反病毒等）。

  2. 免费加入大家不珍惜，开始时免费邀请的同学进入后几乎没什么贡献，仔细想想大家是不是这样呢？
  3. 投资自己才是最好的投资，如果助安社区真能帮助你升职、加薪和迎娶白富美（开玩笑），回过头来你会发现，这绝对是你最值得的一笔投资 ！

¶嘉宾入驻

已入职大厂（甲方、乙方都可以）的同学可 免费加入 ，但必须在星球中发布一个相对完整的自我介绍，分享个人的经验和经历（权力越大能力越大），扫码添加微信，备注“大厂”
